# Book Management Web Application

This project showcases my capabilities in building a web application using **Spring Boot** on the back-end and **Angular** on the front-end.

It is a basic book application where users can search for books, add books they have read or plan on reading, and calculate the average rating of a specific book. The front-end is intentionally very simple, with the sole purpose of demonstrating the functionalities of the web application.

## Features

- User account creation
- Email verification and authentication
- User login
- Book search
- Creation of **userBooks** (books read or planned by the user)
- Deletion of **userBooks**
- Average rating calculation for a book


## Authentication method

The application uses **JWT authentication** tokens for all of the requests to validate user identity.

## Back-End & API

The application includes a custom back-end API that allows new books to be added to the database by simply providing the book name.  
This API automatically retrieves book information such as description, cover image, and author by integrating with the **Google Books API**.
